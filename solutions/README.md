# Workshop reference solutions

This folder contains completed versions of the Copilot customizations that participants create during the workshop.

Use these files only as a fallback when a participant gets stuck or needs to compare their result with a known-good solution.

## How to use

- Active starter customizations live under the repository root `.github/` folder.
- Completed reference customizations live under `solutions/.github/`.
- To use a reference solution, copy the relevant file from `solutions/.github/` into the matching path under `.github/`.

Examples:

- Copy `solutions/.github/copilot-instructions.md` to `.github/copilot-instructions.md`.
- Copy `solutions/.github/agents/security-reviewer.agent.md` to `.github/agents/security-reviewer.agent.md`.
- Copy `solutions/.github/prompts/full-solution-modernization.prompt.md` to `.github/prompts/full-solution-modernization.prompt.md`.

Do not keep all reference solutions active at the start of the workshop. Participants should create most customizations themselves.
