Operations notes

Routing rules are tribal knowledge:

  • VPN goes to Networking unless user mentions password.
  • Payroll or finance-impacting incidents become Critical.
  • Hardware requests usually go to Desktop.

The handoff report is generated by the Python script in this folder.
